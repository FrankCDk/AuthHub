# AuthHub

AuthHub es una aplicación para la gestión de autenticación y autorización de usuarios.

## Características

- Registro de usuarios
- Inicio de sesión
- Recuperación de contraseña
- Gestión de roles y permisos

## Requisitos

- Node.js
- MongoDB

## Instalación

1. Clona el repositorio:
    ```bash
    git clone https://github.com/tu-usuario/AuthHub.git
    ```
2. Navega al directorio del proyecto:
    ```bash
    cd AuthHub
    ```
3. Instala las dependencias:
    ```bash
    npm install
    ```

## Uso

1. Inicia el servidor:
    ```bash
    npm start
    ```
2. Abre tu navegador y navega a `http://localhost:3000`
